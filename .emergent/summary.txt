-   **System Tray Application:** The use of  to create a Python application that runs in the Windows system tray with a status icon and menu.
-   **Native OS Notifications:** Use of  to send native Windows notifications from the Python script.
-   **Role-Based Access Control (RBAC):** A detailed permission system was built into the frontend, allowing for default permissions based on a user's role, with the ability for an Admin to grant specific overrides (special permissions).
