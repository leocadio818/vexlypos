**what currently exists?**
A sophisticated Restaurant POS system with a FastAPI backend and a React frontend. The application features a comprehensive Inventario Maestro (Inventory Master) module which includes:
-   **Core Inventory Management:** Full CRUD for ingredients, suppliers, warehouses, and recipes.
-   **Advanced Unit & Conversion Management:** A system to define custom units of measure (e.g., Box, Bottle) and calculate conversion factors (e.g., 1 Box = 24 Units). Includes an in-modal calculator for ease of use.
-   **Universal Conversion Logic:** A feature that allows a single ingredient to be linked to multiple saleable products with different consumption units, all while tracking back to a single inventory stock. A dedicated Analysis view shows this relationship and calculates costs dynamically.
-   **Multi-Level Stock Visualization:** The main stock view displays inventory in a hierarchical format (e.g., ) for intuitive, at-a-glance counting.
-   **Stock Difference Management:** A streamlined process for adjusting inventory for Differences (formerly Waste), such as physical count discrepancies or damaged goods, with full audit logging.
-   **Financial Reporting & Auditing:**
    -   **Inventory Valuation Report:** A dedicated report showing the total financial value of stock, broken down by category and warehouse, and highlighting dead stock (high-value, low-rotation items).
    -   **Ingredient Audit History:** A detailed, filterable, and exportable log of every change made to an ingredient (cost, conversion factor, etc.).
-   **Order & Sales Flow:** A complete order screen, advanced item cancellation/void system with manager PIN authorization, and reporting.
